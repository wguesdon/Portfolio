# BCR Repertoire Analysis — COVID-19

End-to-end bulk B-cell receptor (BCR) repertoire pipeline using
[nf-core/airrflow](https://nf-co.re/airrflow), applied to COVID-19 patients
with varying disease severity.

**Dataset:** [PRJNA639956](https://www.ncbi.nlm.nih.gov/bioproject/PRJNA639956) —
Galson et al. 2020, *Frontiers in Immunology*

---

## Biological Context

Galson et al. performed deep amplicon-based IGH sequencing of B-cell receptor
repertoires from COVID-19 patients across the severity spectrum, revealing convergent
immune signatures — the same antibody sequences independently arising in multiple
patients, suggesting a shared adaptive response to SARS-CoV-2.

| Attribute | Value |
|-----------|-------|
| Organism | *Homo sapiens* |
| Locus | IGH (IgG heavy chain) |
| Protocol | Amplicon-based (NEBNext Immune Sequencing Kit) |
| Sequencing | Illumina MiSeq 2×300 bp |
| Cohort | 31 COVID-19 (mild/moderate/severe/critical) + 6 healthy controls |
| SRA Project | [PRJNA639956](https://www.ncbi.nlm.nih.gov/bioproject/PRJNA639956) |

## Pipeline Overview

```
FASTQ (paired-end, 2×300 bp)
        │
        ▼
┌──────────────────────────────────────┐
│  pRESTO                              │
│  • UMI consensus building            │
│  • Primer trimming (C-region + V)    │
│  • Quality filtering                 │
│  • Paired-end assembly               │
└──────────────┬───────────────────────┘
               │  assembled sequences (FASTA/FASTQ)
               ▼
┌──────────────────────────────────────┐
│  IgBLAST (IMGT reference)           │
│  • V/D/J gene assignment             │
│  • CDR/FWR annotation                │
│  • Junction extraction               │
└──────────────┬───────────────────────┘
               │  AIRR-formatted TSV
               ▼
┌──────────────────────────────────────┐
│  Immcantation (Change-O / Alakazam)  │
│  • Clonotype grouping (SCOPer)       │
│  • Somatic hypermutation (SHM)       │
│  • V-gene usage                      │
│  • CDR3 length distribution          │
│  • Clonal diversity metrics          │
└──────────────────────────────────────┘
               │  R reports + TSV outputs
               ▼
         results/airrflow/galson2020/
```

## Quick Start

### Prerequisites

- [Nextflow](https://nextflow.io/) ≥ 24.x
- Docker (local) **or** AWS credentials + `S3_BUCKET` (cloud)
- sra-tools ≥ 3.0 for data download (`conda install -c bioconda sra-tools`)

### 1. Install Nextflow

```bash
make install-nextflow
```

### 2. Download data

```bash
conda activate sra-tools
make data-galson2020
```

Downloads ~6 paired-end FASTQ pairs (~2–3 GB) from SRA and writes
`data/galson2020/samplesheet.csv`.

### 3a. Run locally (Docker)

```bash
make run-airrflow-galson2020
```

### 3b. Run on AWS Batch (Spot)

```bash
export S3_BUCKET=your-bucket-name
make upload-galson2020
ENV=aws S3_BUCKET=$S3_BUCKET make run-airrflow-galson2020
```

### Preview the command without running

```bash
make dry-run-galson2020
```

### Resume a run

```bash
./scripts/run.sh --dataset galson2020 --resume
```

## Parameters

Key parameters in `pipelines/airrflow/params/galson2020.json`:

| Parameter | Value | Description |
|-----------|-------|-------------|
| `library_generation_method` | `specific_pcr_umi` | Amplicon with UMI (NEBNext kit) |
| `cprimer_position` | `R1` | C-region primer on Read 1 |
| `umi_length` | `8` | UMI length in bp |
| `umi_position` | `R2` | UMI on Read 2 |
| `reassign` | `true` | Re-annotate with IgBLAST |
| `productive_only` | `true` | Keep only productive rearrangements |
| `threshold_method` | `gmm` | Gaussian mixture model for clone threshold |
| `clonal_threshold` | `auto` | Auto-detect from data |

> **Note:** Verify `library_generation_method` and UMI parameters against the
> exact protocol described in the paper's Methods section before a production run.

## AWS Architecture

Same compute environment as the RNA-seq project (`nextflow-ce`):

```
Local machine (Nextflow head node)
        │
        ▼
  AWS Batch (nextflow-queue)
  • MANAGED SPOT, c5/m5 instances
  • 0–256 vCPU, scales to zero
        │
        ▼
  S3 Bucket (via Fusion FUSE)
  ├── data/galson2020/       ← FASTQs + samplesheet
  ├── work/                  ← Nextflow work dir
  └── results/airrflow/      ← final outputs
```
