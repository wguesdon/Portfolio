# Bioinformatics Job Market Analysis

Analysis of bioinformatics job postings to identify trends in required skills, tools, qualifications, and salary ranges.

## Project Structure

```
├── data/          # Raw and processed job posting data
├── src/           # Analysis scripts and utilities
├── output/        # Generated figures and tables
├── report.qmd     # Quarto report
└── pyproject.toml
```

## Setup

```bash
uv sync
```
